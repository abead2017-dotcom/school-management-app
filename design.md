# تصميم تطبيق إدارة المدارس - School Management System Design

## نظرة عامة
تطبيق جوال متكامل لإدارة المدارس مع لوحات تحكم مستقلة لكل فئة من المستخدمين (مدير، معلم، طالب، ولي أمر). يعتمد على تسجيل الدخول بالرقم الوطني والمصادقة عبر JWT.

---

## 📱 قائمة الشاشات (Screen List)

### الشاشات المشتركة
1. **شاشة تسجيل الدخول** (Login Screen)
   - إدخال الرقم الوطني
   - إدخال كلمة المرور
   - زر تسجيل الدخول
   - رابط "نسيت كلمة المرور"

2. **شاشة إعادة تعيين كلمة المرور** (Password Reset)
   - إدخال الرقم الوطني
   - إدخال كلمة المرور الجديدة
   - تأكيد كلمة المرور

3. **شاشة الملف الشخصي** (Profile Screen)
   - عرض البيانات الشخصية
   - تعديل البيانات
   - تغيير كلمة المرور
   - تسجيل الخروج

### لوحة تحكم المدير (Admin Dashboard)
1. **الشاشة الرئيسية للمدير** (Admin Home)
   - إحصائيات عامة (عدد الطلاب، المعلمين، الفصول)
   - الإشعارات الأخيرة
   - الأنشطة الأخيرة

2. **إدارة المعلمين** (Teachers Management)
   - قائمة المعلمين
   - إضافة معلم جديد
   - تعديل بيانات المعلم
   - حذف معلم

3. **إدارة الطلاب** (Students Management)
   - قائمة الطلاب
   - إضافة طالب جديد
   - تعديل بيانات الطالب
   - حذف طالب
   - ربط الطالب بولي أمر

4. **إدارة الفصول** (Classes Management)
   - قائمة الفصول
   - إنشاء فصل جديد
   - تعديل الفصل
   - إضافة طلاب للفصل
   - حذف فصل

5. **إدارة المواد** (Subjects Management)
   - قائمة المواد
   - إضافة مادة جديدة
   - تعديل المادة
   - حذف مادة

6. **إدارة الجداول** (Timetables Management)
   - إنشاء جدول دراسي
   - تعديل الجدول
   - ربط المعلمين والمواد بالجدول

7. **إدارة الامتحانات** (Exams Management)
   - قائمة الامتحانات
   - إنشاء امتحان جديد
   - تحديد مواعيد الامتحانات
   - إدارة درجات الامتحانات

8. **إدارة الرسوم الدراسية** (Fees Management)
   - قائمة الرسوم
   - إضافة رسم جديد
   - متابعة المدفوعات
   - إرسال إشعارات بالرسوم المتأخرة

9. **إدارة الإشعارات** (Notifications Management)
   - إرسال إشعار عام
   - إرسال إشعار مخصص
   - قائمة الإشعارات المرسلة

10. **التقارير والإحصائيات** (Reports & Statistics)
    - تقرير الأداء العام
    - نسب النجاح السنوية
    - تقارير الحضور

### لوحة تحكم المعلم (Teacher Dashboard)

## 🔄 مسارات المستخدم الرئيسية (Key User Flows)

### مسار تسجيل الدخول
1. المستخدم يفتح التطبيق
2. يدخل الرقم الوطني وكلمة المرور
3. يضغط على زر "تسجيل الدخول"
4. التطبيق يتحقق من البيانات عبر API
5. إذا كانت البيانات صحيحة، يتم إنشاء JWT token
6. يتم حفظ الـ token في التخزين الآمن
7. يتم توجيه المستخدم إلى لوحة التحكم الخاصة به

### مسار مدير يضيف معلم جديد
1. المدير يفتح قائمة المعلمين
2. يضغط على زر "إضافة معلم جديد"
3. يملأ نموذج البيانات (الرقم الوطني، الاسم، البريد الإلكتروني، إلخ)
4. يضغط على زر "حفظ"
5. يتم إرسال البيانات إلى API
6. يتم إنشاء حساب المعلم في قاعدة البيانات
7. يتم عرض رسالة تأكيد النجاح

### مسار معلم يسجل درجات الطلاب
1. المعلم يفتح قائمة الفصول
2. يختار الفصل المطلوب
3. يختار المادة
4. يرى قائمة الطلاب مع حقول إدخال الدرجات
5. يدخل درجات الطلاب
6. يضغط على زر "حفظ"
7. يتم إرسال الدرجات إلى API
8. يتم حفظ الدرجات في قاعدة البيانات

### مسار طالب يعرض درجاته
1. الطالب يفتح التطبيق
2. يذهب إلى قسم "الدرجات"
3. يرى قائمة المواد مع درجاته
4. يمكنه الضغط على المادة لرؤية التفاصيل
5. يرى المعدل العام والتقييمات

### مسار ولي أمر يتابع ابنه
1. ولي الأمر يفتح التطبيق
2. يختار الابن من قائمة الأبناء
3. يرى الجدول الدراسي والواجبات والدرجات
4. يمكنه الضغط على أي عنصر لرؤية التفاصيل
5. يرى حالة الرسوم والمدفوعات

## 📊 نموذج البيانات (Data Model)

### الجداول الرئيسية

#### جدول المستخدمين (Users)
- `id` - معرف فريد
- `nationalId` - الرقم الوطني (فريد)
- `password` - كلمة المرور (مشفرة)
- `name` - الاسم الكامل
- `email` - البريد الإلكتروني
- `phone` - رقم الهاتف
- `role` - نوع المستخدم (admin, teacher, student, parent)
- `createdAt` - تاريخ الإنشاء
- `updatedAt` - تاريخ التحديث

#### جدول المعلمين (Teachers)
- `id` - معرف فريد
- `userId` - معرف المستخدم
- `specialization` - التخصص
- `qualification` - المؤهل العلمي
- `joinDate` - تاريخ الانضمام

#### جدول الطلاب (Students)
- `id` - معرف فريد
- `userId` - معرف المستخدم
- `classId` - معرف الفصل
- `parentId` - معرف ولي الأمر
- `enrollmentDate` - تاريخ الالتحاق

#### جدول الفصول (Classes)
- `id` - معرف فريد
- `name` - اسم الفصل
- `level` - المستوى الدراسي
- `capacity` - السعة
- `createdAt` - تاريخ الإنشاء

#### جدول المواد (Subjects)
- `id` - معرف فريد
- `name` - اسم المادة
- `code` - رمز المادة
- `description` - الوصف

#### جدول الجداول الدراسية (Timetables)
- `id` - معرف فريد
- `classId` - معرف الفصل
- `day` - اليوم
- `startTime` - وقت البداية
- `endTime` - وقت النهاية
- `subjectId` - معرف المادة
- `teacherId` - معرف المعلم

#### جدول الواجبات (Assignments)
- `id` - معرف فريد
- `classId` - معرف الفصل
- `subjectId` - معرف المادة
- `teacherId` - معرف المعلم
- `title` - عنوان الواجب
- `description` - الوصف
- `dueDate` - تاريخ التسليم
- `createdAt` - تاريخ الإنشاء

#### جدول الدرجات (Grades)
- `id` - معرف فريد
- `studentId` - معرف الطالب
- `subjectId` - معرف المادة
- `grade` - الدرجة
- `semester` - الفصل الدراسي
- `academicYear` - السنة الدراسية

#### جدول الامتحانات (Exams)
- `id` - معرف فريد
- `classId` - معرف الفصل
- `subjectId` - معرف المادة
- `date` - تاريخ الامتحان
- `time` - وقت الامتحان
- `duration` - المدة
- `totalMarks` - الدرجة الكلية

#### جدول الحضور (Attendance)
- `id` - معرف فريد
- `studentId` - معرف الطالب
- `date` - التاريخ
- `status` - الحالة (present, absent, late)

#### جدول الرسوم (Fees)
- `id` - معرف فريد
- `studentId` - معرف الطالب
- `amount` - المبلغ
- `dueDate` - تاريخ الاستحقاق
- `paidDate` - تاريخ الدفع
- `status` - الحالة (pending, paid, overdue)

#### جدول الإشعارات (Notifications)
- `id` - معرف فريد
- `userId` - معرف المستخدم
- `title` - العنوان
- `message` - الرسالة
- `type` - نوع الإشعار
- `read` - هل تم قراءته
- `createdAt` - تاريخ الإنشاء

## 🎯 المتطلبات الوظيفية

### المصادقة والأمان
- تسجيل الدخول بالرقم الوطني وكلمة المرور
- تشفير كلمات المرور باستخدام bcrypt
- إنشاء JWT tokens للمصادقة
- إعادة تعيين كلمة المرور
- تسجيل الخروج

### لوحة تحكم المدير
- إدارة المعلمين (إضافة، تعديل، حذف)
- إدارة الطلاب (إضافة، تعديل، حذف)
- إدارة الفصول (إنشاء، تعديل، حذف)
- إدارة المواد (إضافة، تعديل، حذف)
- إدارة الجداول الدراسية
- إدارة الامتحانات
- إدارة الرسوم والمدفوعات
- إرسال الإشعارات
- عرض التقارير والإحصائيات

### لوحة تحكم المعلم
- عرض الجداول اليومية والأسبوعية
- إضافة الواجبات
- تسجيل الدرجات
- متابعة الحضور
- التواصل مع الإدارة والطلاب

### لوحة تحكم الطالب
- عرض الجداول الدراسية
- عرض الواجبات
- عرض الدرجات والنتائج
- عرض مواعيد الامتحانات
- متابعة الرسوم الدراسية
- عرض سجل الحضور

### لوحة تحكم ولي الأمر
- متابعة درجات الأبناء
- متابعة الحضور
- متابعة الواجبات
- متابعة الرسوم والمدفوعات
- استقبال الإشعارات

## 📐 معايير التصميم

### مبادئ التصميم
- **البساطة**: واجهة بسيطة وسهلة الاستخدام
- **الوضوح**: معلومات واضحة ومنظمة
- **الاستجابة**: تطبيق متجاوب لجميع أحجام الشاشات
- **الأداء**: تطبيق سريع وفعال
- **الأمان**: حماية البيانات والخصوصية

### معايير الوصول
- دعم الخطوط الكبيرة
- تباين عالي للنصوص
- دعم قارئات الشاشة

### التوافقية
- iOS 13+
- Android 8+
- دعم الوضع الليلي والنهاري
